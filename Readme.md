## :octocat:Clone this repo

1. Clone :

```sh
  $ git clone https://github.com/VictorGabrielMS/uiclone_twitch.git
```

2. Execute:

```sh
  $ cd uiclone_twitch
```

3. Execute:

```sh
  $ npm install
  $ npm run start
```
